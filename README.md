## Git Branch
### master branch
최종 브랜치

### develop branch
feature branch를 merge하는 브랜치
최종적으로 버그가 수정되면 master branch에 merge함

### feature branch
새로운 기능 개발 및 사소한 버그 수정을 위한 브랜치
feature branch에서 작업한 후 develop branch에 merge함
[feature/기능요약] 형식을 따름
